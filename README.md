Смысл такой:

main.py - точка запуска. Здесь вы задаете сценарий: начальное положение, скорость, курс, импульс поворота, длительность моделирования.

config.py - параметры модели: шаг времени, длительность, скорость, коэффициент затухания, максимальная угловая скорость.

models/ship_state.py - описание состояния судна:

x, y, speed, course, yaw_rate

simulation/kinematics.py - формулы движения:

x_next
y_next
course_next
yaw_rate_next

simulation/controls.py - управляющие воздействия: импульс поворота, команда руля, изменение скорости.

visualization/animation.py - вся логика отрисовки и анимации.

notebooks/experiments.ipynb - только для экспериментов, проверки гипотез, графиков и быстрых тестов.
