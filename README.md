# Математический аппарат расчета маневра уклонения от столкновения

## 1. Постановка задачи

В модели рассматривается задача прогнозного проигрывания маневра расхождения собственного судна с судном-целью в двумерной системе координат (X/Y).

Координаты задаются в морских милях, скорость - в узлах, время - в секундах. Ось (X) направлена на восток, ось (Y) направлена на север. Курс задается в навигационной системе: (0^\circ) - север, (90^\circ) - восток, (180^\circ) - юг, (270^\circ) - запад.

Судно-цель считается равномерно и прямолинейно движущимся объектом. Собственное судно может изменять курс и скорость с учетом ограничений по скорости поворота и ускорению.

## 2. Кинематическая модель движения

Для курса (K) единичный вектор направления движения имеет вид:

[
\mathbf{e}(K)=
\begin{bmatrix}
\sin K \
\cos K
\end{bmatrix}
]

где (K) переводится из градусов в радианы.

Если скорость судна равна (V), то вектор скорости:

[
\mathbf{v}=V\mathbf{e}(K)=
\begin{bmatrix}
V\sin K \
V\cos K
\end{bmatrix}
]

С учетом течения:

[
\mathbf{v}_g=\mathbf{v}+\mathbf{v}_c
]

где:

[
\mathbf{v}*c=
\begin{bmatrix}
V*{c,x} \
V_{c,y}
\end{bmatrix}
]

Поскольку скорость задана в узлах, а время - в секундах, шаг интегрирования в часах:

[
\Delta t_h=\frac{\Delta t}{3600}
]

Положение судна на следующем шаге:

[
\mathbf{r}_{k+1}=\mathbf{r}*k+\mathbf{v}*{g,k}\Delta t_h
]

где:

[
\mathbf{r}_k=
\begin{bmatrix}
x_k \
y_k
\end{bmatrix}
]

## 3. Модель судна-цели

Судно-цель движется без маневрирования:

[
K_t=const
]

[
V_t=const
]

Тогда его положение рассчитывается по формуле:

[
\mathbf{r}*{t,k+1}=\mathbf{r}*{t,k}+\left(V_t\mathbf{e}(K_t)+\mathbf{v}_c\right)\Delta t_h
]

## 4. Модель собственного судна

Собственное судно управляется через заданный курс (K_{cmd}) и заданную скорость (V_{cmd}).

Ошибка курса:

[
e_K=wrap(K_{cmd}-K)
]

где функция (wrap) приводит ошибку курса к диапазону:

[
[-180^\circ,180^\circ]
]

Желаемая угловая скорость:

[
\omega_{des}=k_K e_K
]

Фактическая угловая скорость ограничивается:

[
\omega=clamp(\omega_{des},-\omega_{max},\omega_{max})
]

Изменение курса за шаг:

[
\Delta K=\omega \Delta t
]

Новый курс:

[
K_{k+1}=normalize(K_k+\Delta K)
]

Ошибка скорости:

[
e_V=V_{cmd}-V
]

Изменение скорости за шаг:

[
\Delta V=clamp(e_V,-a_{max}\Delta t,a_{max}\Delta t)
]

Новая скорость:

[
V_{k+1}=V_k+\Delta V
]

После обновления курса и скорости положение собственного судна рассчитывается по кинематической модели.

## 5. Расчет CPA, DCPA и TCPA

Для оценки опасности используется относительное движение.

Положение собственного судна:

[
\mathbf{r}_o
]

Положение судна-цели:

[
\mathbf{r}_t
]

Относительное положение цели относительно собственного судна:

[
\mathbf{r}_{rel}=\mathbf{r}_t-\mathbf{r}_o
]

Скорости судов:

[
\mathbf{v}_o=V_o\mathbf{e}(K_o)
]

[
\mathbf{v}_t=V_t\mathbf{e}(K_t)
]

Относительная скорость:

[
\mathbf{v}_{rel}=\mathbf{v}_t-\mathbf{v}_o
]

Если относительная скорость близка к нулю:

[
|\mathbf{v}_{rel}|^2 \approx 0
]

то (TCPA) принимается бесконечным, а (DCPA) равен текущей дистанции.

В общем случае время до кратчайшего сближения:

[
t_{CPA,h}=
-\frac{\mathbf{r}*{rel}\cdot\mathbf{v}*{rel}}
{|\mathbf{v}_{rel}|^2}
]

В секундах:

[
TCPA=3600t_{CPA,h}
]

Положение судов в момент кратчайшего сближения:

[
\mathbf{r}_{o,CPA}=\mathbf{r}_o+\mathbf{v}*o t*{CPA,h}
]

[
\mathbf{r}_{t,CPA}=\mathbf{r}_t+\mathbf{v}*t t*{CPA,h}
]

Дистанция кратчайшего сближения:

[
DCPA=|\mathbf{r}*{t,CPA}-\mathbf{r}*{o,CPA}|
]

Текущая дистанция:

[
D=|\mathbf{r}_t-\mathbf{r}_o|
]

## 6. Условие опасного сближения

Маневр активируется только при одновременном выполнении условий:

[
D \leq D_{activation}
]

[
DCPA < D_{safe}
]

[
0 < TCPA < T_{limit}
]

где:

* (D_{safe}=0.5) морской мили - безопасная дистанция;
* (D_{activation}=3.0) морские мили - дистанция активации маневра;
* (T_{limit}=900) секунд - временной предел опасного сближения.

Таким образом, одного геометрического пересечения линий пути недостаточно. Маневр требуется только тогда, когда прогноз относительного движения показывает опасное сближение во времени.

## 7. Пространство допустимых маневров

Алгоритм рассматривает конечное множество вариантов изменения курса и скорости.

Множество изменений курса:

[
\Delta K \in {-30^\circ,-25^\circ,...,0^\circ,...,+25^\circ,+30^\circ}
]

Множество изменений скорости:

[
\Delta V \in {-6,-5,...,0,...,+5,+6}
]

Каждый кандидат задается парой:

[
u_i=(\Delta K_i,\Delta V_i)
]

Кандидатный курс уклонения:

[
K_{avoid}=normalize(K_0+\Delta K_i)
]

Кандидатная скорость уклонения:

[
V_{avoid}=V_0+\Delta V_i
]

Кандидат отбрасывается, если:

[
V_{avoid}<V_{min}
]

## 8. Проверка безопасности кандидата

Для каждого кандидата выполняется численное моделирование движения судов на оставшемся горизонте прогноза.

На каждом шаге рассчитывается дистанция:

[
D_k=|\mathbf{r}*{t,k}-\mathbf{r}*{o,k}|
]

Кандидат считается недопустимым, если хотя бы на одном шаге:

[
D_k < D_{safe}
]

Кандидат считается безопасным, если:

[
D_k \geq D_{safe}
]

для всех шагов моделирования.

Множество безопасных кандидатов:

[
\mathcal{F}=
\left{
u_i \in \mathcal{U}
\mid
\min_k D_k(u_i) \geq D_{safe}
\right}
]

где:

* (\mathcal{U}) - множество всех проверяемых кандидатов;
* (\mathcal{F}) - множество безопасных кандидатов.

## 9. Функция стоимости маневра

Для каждого безопасного кандидата рассчитывается стоимость:

[
J(u_i)=w_K|\Delta K_i|+w_V|\Delta V_i|
]

В текущей модели:

[
w_K=1
]

[
w_V=2
]

Следовательно:

[
J(u_i)=|\Delta K_i|+2|\Delta V_i|
]

Такой критерий означает, что изменение скорости штрафуется сильнее, чем изменение курса. Алгоритм стремится выбрать безопасный маневр с минимальным отклонением от исходного режима движения.

## 10. Выбор оптимального маневра

Выбранный маневр определяется как:

[
u^*=\arg\min_{u_i \in \mathcal{F}}J(u_i)
]

То есть выбирается безопасный кандидат с минимальной стоимостью.

Если несколько кандидатов имеют одинаковую стоимость, может применяться лексикографическое правило выбора:

[
u^*=
\arg\min_{u_i \in \mathcal{F}}
\left(
J(u_i),
p(u_i),
T_{maneuver}(u_i),
|\Delta K_i|,
|\Delta V_i|
\right)
]

где (p(u_i)) - приоритет правого поворота:

[
p(u_i)=
\begin{cases}
0, & \Delta K_i > 0 \
1, & \Delta K_i < 0
\end{cases}
]

При равной стоимости предпочтение получает правый поворот.

## 11. Возврат на исходную линию пути

Исходная линия пути задается начальной точкой и начальным курсом:

[
\mathbf{r}_0=
\begin{bmatrix}
x_0 \
y_0
\end{bmatrix}
]

[
\mathbf{p}=\mathbf{e}(K_0)
]

Текущий вектор от начальной точки к судну:

[
\mathbf{q}=\mathbf{r}-\mathbf{r}_0
]

Продольная проекция на исходную линию пути:

[
s=\mathbf{q}\cdot\mathbf{p}
]

Поперечное отклонение от линии пути:

[
e_{\perp}=p_x(y-y_0)-p_y(x-x_0)
]

Упреждающая точка на исходной линии пути:

[
\mathbf{r}_{lookahead}=\mathbf{r}_0+\mathbf{p}(s+L)
]

где (L) - расстояние упреждения.

Курс возврата:

[
K_{return}=bearing(\mathbf{r},\mathbf{r}_{lookahead})
]

Начало возврата разрешается при выполнении условий:

[
D>D_{safe}C_{return}
]

[
TCPA<0
]

То есть судно-цель уже прошло точку кратчайшего сближения, а текущая дистанция имеет заданный запас.

Возврат считается завершенным, если:

[
|e_{\perp}| \leq \varepsilon_{track}
]

После этого собственное судно возвращается к начальному курсу и начальной скорости:

[
K_{cmd}=K_0
]

[
V_{cmd}=V_0
]

## 12. Доказательство дискретной оптимальности

Пусть задано конечное множество кандидатов:

[
\mathcal{U}={u_1,u_2,...,u_N}
]

Для каждого кандидата (u_i) алгоритм моделирует движение судов и вычисляет минимальную дистанцию:

[
D_{min}(u_i)=\min_k D_k(u_i)
]

Кандидат является допустимым, если:

[
D_{min}(u_i)\geq D_{safe}
]

Тогда множество допустимых кандидатов:

[
\mathcal{F}=
{u_i \in \mathcal{U}:D_{min}(u_i)\geq D_{safe}}
]

Для каждого допустимого кандидата рассчитывается стоимость:

[
J(u_i)=|\Delta K_i|+2|\Delta V_i|
]

Алгоритм возвращает:

[
u^*=\arg\min_{u_i \in \mathcal{F}}J(u_i)
]

Так как множество (\mathcal{U}) конечно, алгоритм перебирает все его элементы. Следовательно, каждый кандидат либо проверяется и включается в (\mathcal{F}), либо отбрасывается как небезопасный.

Предположим, что существует другой безопасный кандидат (u_j), для которого:

[
J(u_j)<J(u^*)
]

Так как (u_j) безопасен, он должен принадлежать множеству (\mathcal{F}). Так как алгоритм выбирает элемент с минимальным значением (J) среди всех элементов (\mathcal{F}), он должен был выбрать (u_j), а не (u^*). Получаем противоречие.

Следовательно:

[
J(u^*)\leq J(u_i)
]

для всех:

[
u_i \in \mathcal{F}
]

Таким образом, выбранный маневр является оптимальным в пределах заданного дискретного множества допустимых маневров и принятой функции стоимости.

## 13. Границы применимости утверждения об оптимальности

В текущей постановке нельзя утверждать, что алгоритм находит глобально оптимальный маршрут среди всех возможных траекторий.

Глобальная задача требовала бы поиска среди всех возможных функций управления:

[
K_{cmd}(t)
]

[
V_{cmd}(t)
]

или среди всех возможных траекторий:

[
\mathbf{r}_o(t)
]

В данной модели рассматривается только конечное множество простых маневров:

[
(\Delta K,\Delta V)
]

Поэтому оптимальность имеет дискретный и условный характер.

Корректная формулировка:

> В рамках принятой дискретной модели и заданного конечного множества допустимых маневров алгоритм выбирает маневр, оптимальный по функции стоимости при выполнении ограничения безопасной дистанции.

Иными словами, алгоритм не находит глобально оптимальную траекторию в непрерывном пространстве всех возможных маршрутов. Он выбирает лучший безопасный маневр из заранее заданной сетки вариантов изменения курса и скорости.

## 14. Итоговый вывод

Разработанная модель реализует прогнозное проигрывание маневра расхождения на основе кинематической модели движения судов, расчета CPA/DCPA/TCPA и дискретного перебора вариантов изменения курса и скорости собственного судна.

Маневр активируется только при наличии опасного сближения во времени. Каждый кандидатный маневр проверяется численным моделированием на всем горизонте прогноза. Кандидаты, нарушающие безопасную дистанцию, отбрасываются. Среди безопасных кандидатов выбирается маневр с минимальной стоимостью, определяемой взвешенной суммой изменения курса и скорости.

Следовательно, выбранный маневр является оптимальным в пределах заданной дискретной сетки перебора и принятой функции стоимости. Такое утверждение является корректным для текущей baseline-модели и не требует заявления о глобальной оптимальности маршрута в непрерывном пространстве траекторий.
